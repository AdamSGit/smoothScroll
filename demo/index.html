smoothScroll est un plug-in jQuery dont le but est de remplacer la barre de scroll par défaut du navigateur par une barre plus esthétique et personnalisable à souhait en css. Simple d’utilisation et pensée pour vous fournir un paramètrage avancée, elle a l’avantage d’offrir le même rendu sur tous les systèmes d’exploitation.

Légère

Puissante

Cross Browser

Exemple

Vous pouvez voir un exemple directement sur cette page, en scrollant dans la page. La barre de scroll remplace la barre habituelle de votre navigateur.

Utilisation

Comment utiliser le plug-in ? Tout d’abord :

  • Récupérez la dernière version stable de jQuery ici et joignez-la à votre projet
  • Télécharger la dernière version stable ici, en haut à gauche
  • Joignez le au ‹head› votre page html comme ceci (par exemple) :

‹script type="text/javascript" src="js/smoothScroll-1.0.js"›‹/script›

  • Une fois ceci fait, pour initialiser le plug-in, ajoutez ce code (toujours dans le ‹head›)
‹script type="text/javascript" src="js/smoothScroll-1.0.js"›

$(document).load(function() {
$(window).smoothScroll({ param1: value1, param2: value2 });
});

‹/script›
  • Et c’est tout !

Paramètres

theme

Ce paramètre permet de modifier rapidement et simplement l’apparence de la barre.
Valeurs possibles :

  • dark : Sélectionner le thème dark
  • light : Sélectionner le thème light

Valeurs par défaut : dark

autoHide

Ce paramètre permet de masquer automatiquement la barre lorsqu’elle est inactive.
Valeurs possibles :

  • true : Masque la barre
  • false : Désactive cette option

Valeurs par défaut : false

opacity

Ce paramètre permet de modifier la transparence de la barre.
Valeurs possibles :

  • un chiffre de type 0, 1 ou 0,x entre 0 et 1

Valeurs par défaut : 0.6

scrollSpeed

Ce paramètre permet de modifier la vitesse de scroll.
Valeurs possibles :

  • 1 : Vitesse de défilement douce
  • 2 : Vitesse de défilement normale
  • 3 : Vitesse de défilement rapide

Valeurs par défaut : 2 (normal)

animate

Ce paramètre permet de rendre le déplacelement de la barre plus fluide en l'animant. Note : Impossible à reproduire lors d’un usage normal, mais lors d'un scroll rapide, intensif et long, on peut observer la création d’un léger décalage sur l'animation.
Valeurs possibles :

  • true : anime la barre
  • false : n’anime pas la barre

Valeurs par défaut : true

background

Ce paramètre permet d’afficher ou de masquer le fond de la barre
Valeurs possibles :

  • true : affiche le fond
  • false : masque le fond

Valeurs par défaut : false (fond masqué)

delayHideBar

Ce paramètre permet de modifier le délai de disparition de la barre dans le cas où autoHide vaut true
Valeurs possibles :

  • un nombre en millisecondes (ex : 1000 pour 1 seconde de délai)

Valeurs par défaut : 1000 (une seconde)

cssBar

Ce paramètre vous permet un paramètrage visuel de la barre plus poussé en ajoutant toutes les propriétés CSS désirées
Valeurs possibles :

  • Un tableau associatif contenant des propriétés CSS. Exemple :

cssBar: { background: ‘blue’, borderRadius: ‘4px’, border: ‘1px solid #fff’ }

Valeurs par défaut : false

cssBg

Identique à cssBar, mais pour le fond de la barre (masqué par défaut). Pour que les propriétés CSS soient prises en compte, vous devez au préalable afficher le fond avec le paramètre background
Valeurs possibles :

  • Un tableau associatif contenant des propriétés CSS.

Valeurs par défaut : false

FAQ

J’appelle mes pages en ajax avec différentes hauteurs / mes pages peuvent changer de hauteur en cours de route, est ce un problème ?

Bien sûr que non, le plug in va détecter automatiquement tout changement de résolution verticale.

J’utilise d’autres système / plugs-in en javascript qui (effectuent un scroll dans | qui déplace) la page, la barre ne rentrera pas en conflit ?

Non plus, la barre va s'animer normalement, aucun conflit n'aura lieu. Notez que le déplacement horizontal ne provoque aucune réaction de la barre.
